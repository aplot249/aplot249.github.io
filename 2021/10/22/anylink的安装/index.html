anylink的安装

Created2021-10-22|Updated2021-10-23
|Post View:

拉取anylink代码

从gitee上拉取 git clone https://gitee.com/bjdgyc/anylink.git

安装 golang >= 1.16 和 nodejs >= 14.x

  1. wget https://deb.nodesource.com/setup_16.x 数字16可以更改为其他版本号
  2. bash setup_14.x && apt update && apt install nodejs -y 进行安装
  3. add-apt-repository ppa:longsleep/golang-backports 添加go的ppa源
  4. apt install golang-go -y

    卸载 nodejs

  5. apt remove nodejs 只会卸载 nodejs，但是会保留配置文件，方便以后再次安装 nodejs
  6. apt purge nodejs 如果不想保留配置文件，这将会卸载 nodejs 和其相关的配置文件。
  7. apt autoremove 移除和 nodejs 一起安装但是现在没有被使用的包

    安装certbot

  8. apt install python-certbot-nginx -y --allow-unauthenticated
  9. 生成证书文件 certbot certonly

  10. 备份anylink的配置文件 cd /home/anylink/anylink-deploy/conf/ && cp server-sample.toml server.toml
  11. 把证书文件配置进去
