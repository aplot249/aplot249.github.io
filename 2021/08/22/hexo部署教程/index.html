hexo部署教程

Created2021-08-22|Updated2021-08-23
|Post View:

本地运行

  1. 下载hexo包：npm i hexo -g
  2. 初始化博客网站 hexo init blog
  3. 创建md文件 hexo new xx教程
  4. 编辑器打开source/_posts文件夹下创建的 xx教程.md 文件，进行编辑
  5. 编辑好之后，可以本地运行查看下 hexo server 运行在4000端口上
  6. md文件转静态文件 hexo generate 会创建public文件夹，里面是转好的静态资源

    更换主题

  7. 首先为了博客界面界面美化，可以选择安装好看的主题样式
    此处罗列了一些主题样式
  8. git拉取喜欢的主题到本地，放在themes文件夹下，更改配置文件_config.yml
  9. 启动博客运行查看效果 hexo server

    部署到github

  10. 首先在github上创建仓库，仓库名命名为github用户名.github.io这种形式
  11. 打包本地博客静态资源 hexo generate 此时会生成会创建public文件夹，里面是转好的静态资源
  12. 安装deployer-git npm install hexo-deployer-git --save
  13. 更改配置文件_config.yml，在末尾，更改为
  14. 上传 hexo deploy 顺利的话，等待几秒钟，访问 https://github用户名.github.io 就能看到效果了
